# ElevenPay — Design System

## Design Philosophy

**"Premium fintech first. Football second."**

ElevenPay should not look like a football app. It should look like a premium fintech app with football integrated into it. Think: Apple Wallet (clean), Revolut (modern fintech), Monzo (simple), Stripe Dashboard (professional), with subtle football branding.

**Avoid:** grass backgrounds, giant footballs, neon stadium graphics, cartoon UI.

**Instead use:** glassmorphism, rounded cards, clean typography, smooth animations, football icons only where meaningful.

## Color Palette

- Background: `#0B0F14` (dark charcoal)
- Primary: `#00C853` (emerald green)
- Secondary: `#1E293B` (dark slate)
- Accent: `#FFD54F` (gold — for rewards and wins)
- Success: `#00E676`
- Error: `#FF5252`
- Card: `#161B22`
- Text Primary: `#FFFFFF`
- Text Secondary: `#94A3B8`

## Typography

Font: **Inter**. Weights: Bold, SemiBold, Medium, Regular. No fancy fonts.

## Icons

**Lucide Icons.** Examples: Wallet, Dollar, QR, Football, Users, Map, Clock, Sparkles, Robot.

## Navigation

Bottom navigation, five tabs only: **Home, Wallet, Watch, AI, Profile**.

## Micro Animations

- Wallet balance counts up.
- Transaction slides in.
- Payment success: green check animation.
- Prediction winner: confetti.
- Pool joined: card expands.

## Border Radius

- Cards: 24
- Buttons: 18
- Inputs: 16

## Shadows

Very soft. Don't overdo them.

## Spacing

8-point grid: 8, 16, 24, 32, 48.

## Reusable Components

WalletCard, MatchCard, PoolCard, ActionButton, TransactionCard, GlassCard, AIMessage, UserMessage, Avatar, BottomSheet, PrimaryButton, SecondaryButton.

## Overall Feel

Imagine if Revolut built a wallet exclusively for football fans during the World Cup. That's the aesthetic.
